    Project Details

    A brief overview of the projects included above.

    Project 1: Data Wrangling

    In this project, I analyzed movie box office trends from 2000 to 2024 to better understand how domestic and foreign markets, peak revenue years, and inflation‑adjusted ticket prices shape industry performance. The results show that foreign markets have become increasingly important, average worldwide revenue peaked in 2019 before declining during the pandemic, and rising ticket prices rather than increased attendance have driven much of the recent revenue growth. Together, these findings highlight how global audiences and inflation pressures influence the true state of the movie industry. I did this project in Python using libraries such as Pandas and Matplotlib for data manipulation and visualization.

    Project 2: Baseball SQL Analysis

    In this project, I applied SQL queries to analyze baseball statistics. I built queries to uncover player performance trends, team comparisons, and historical insights. The analysis showcases my ability to design efficient queries and interpret results in a clear, meaningful way. I did this project using SQL to interact with a baseball statistics database and extract relevant information.
